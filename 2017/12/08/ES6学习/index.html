let 与 var

  1. 作用域只局限于当前代码块

  2. 声明的变量作用域不会被提升

  3. 在相同的作用域下不能声明相同的变量

  4. for 循环提现let的作用域

const

  1. 作用域只局限于当前代码块

  2. 声明的变量作用域不会被提升

  3. 不能重复声明

  4. 声明的变量必须赋值

  5. const声明的变量一般不能修改，引用类型除外
